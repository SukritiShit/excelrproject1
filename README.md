# excelrproject1
Project Overview:
This project demonstrates the complete Procure-to-Pay (P2P) process in SAP MM. It covers the full procurement lifecycle from purchase requisition to vendor payment.

Objective:
To implement and understand an end-to-end procurement process using SAP ERP, ensuring efficiency, accuracy, and integration with financial accounting.

Process Flow:
Purchase Requisition → Purchase Order → Goods Receipt → Invoice → Payment

Modules Used:
SAP MM (Material Management)
SAP FI (Financial Accounting)

Steps Involved:
Purchase Requisition (ME51N)
Purchase Order Creation (ME21N)
Goods Receipt (MIGO)
Invoice Verification (MIRO)
Payment Processing (F110)

Tech Stack:
SAP ERP
SAP GUI
GitHub

Key Features:
End-to-end procurement cycle
Real-time inventory updates
Integration with finance module
Accurate invoice verification

Future Improvements:
Workflow automation
SAP S/4HANA integration
Advanced reporting dashboards
